##requirements:
you sould install [tweepy](http://www.tweepy.org/) 
```
pip3 install tweepy
```
and [TextBlob](https://textblob.readthedocs.io/en/dev/)
```
pip3 install -U textblob
```

After you install them you can use the script
```
python3 twitterAnalyzing.py
```
